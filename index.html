Introduction

This website is provided to assist in the development and review of NTCIP and related ISO documents.

Conventions

The text within this website is managed using ReqView to ensure proper traceability is maintained during development. The website is a tool to assist in review, but the target is the text of standards documents. As such, there are some anomalies on the website that are intended to assist in producing the target documents accurately. These items include:

  • References to sections and figures are denoted with !~Section ##~! or !~Figure ##~! where the ## indicates the table row identifier. This text is translated into a sequential reference number in the final document.

Navigation

The major documents on the website are shown in the navigation bar in the upper right. These documents are supplemented with additional materials that are accessible via the Other dropdown menu and include the representations of MIBs and other documents that are used to capture traceability to design elements.

Once the desired document is selected, the Table of contents on the left side of the display is populated so that you can more easily navigate the rather large files.

The main content of each document is presented in a table format that indicates an identifier for each row as well as other key attributes, including a type field that documents the type of item that the row represents. Another column in the table captures the links used to document traceability. These links allow reviewers to quickly jump to traced elements as needed.
